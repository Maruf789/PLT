PLT
===
A new matrix manipulation language and a compiler for it using OCaml

= Build Routine =

$ cd PLT
$ make

= Test Routine =

$ make test

= Folder Structure =

docs - document
Proposal LRM

src - source code
*.ml scanner.mll parser.mly

scripts - helper scripts
test.sh: test script

samples - sample BuckCal programs
*.bc
